O foco foi no enredo, arte e no início da programação.
Estávamos configurando as ferramentas e praticando o uso dos recursos de cada uma. Tivemos uma dificuldades, pois é o primeiro jogo a ser feito por nós. A partir daqui será foco direto para que o jogo seja pelo menos reconhecido.

Conhecendo as ferramentas e aplicando-os

Usamos a Unity para as configurações do nosso jogo e os recursos necessários.
Além dessa ferramenta, usamos o Aseprite para a arte, essa ferramenta é essencial para
pixel art. A estrutura do enredo nesse momento esta vai sessível, porém ainda não está completa,
ademais está sendo construida em um software chamado Twine, para estrutura de roteiro.
